Lista de Compras


Ésta webapp te ayuda a crear una Lista de Compras. Puedes agregar elementos en tu lista, eliminar y borrar tu listado completo.





    Preentrega 2 - Mohart Blanco Torres
    Comisión 64770

    Descripción: El script crea una Lista de Compras usando input para agregar elementos, y tres botones para agregar, eliminar y limpiar la lista. Se utilizó localStorage para guardar la lista de compras y recuperarla cuando se carga la página.
